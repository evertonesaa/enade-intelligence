# ENADE Intelligence

Institutional system for strategic ENADE exam analysis in Engenharia Química.

## 🎯 Objective

Build a structured intelligence system capable of:

- Mapping historical recurrence of ENADE topics
- Classifying questions by macro area and Bloom level
- Identifying cognitive patterns
- Generating analytics dashboards
- Supporting institutional decision-making

---

## 🧱 Architecture

ENADE Intelligence is divided into five layers:

1. Data Foundation
2. Classification Engine
3. Analytics Engine
4. Dashboard
5. Pedagogical Intelligence

---

## 📊 Data Scope

Currently covering:

- ENADE 2011
- ENADE 2014
- ENADE 2017
- ENADE 2019
- ENADE 2023
